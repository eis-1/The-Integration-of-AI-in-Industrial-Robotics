## How to present

1. Open `PROJECT OF Discrete Math.pptx` in PowerPoint (or Google Slides / LibreOffice Impress).
2. If fonts/layout shift:
   - Use PowerPoint "Design" → "Fonts" to pick a close match, or
   - Export to PDF from PowerPoint and present the PDF.
3. Keep the report ready (`Report - Google Docs.pdf`) for references during Q&A.
